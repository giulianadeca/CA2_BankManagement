# CA2 - Bank Management System

## Project Description
Bank Management System developed for CCT College's Algorithms & Constructs and Software Development Fundamentals modules.

## Features
- Employee hierarchy management using Binary Search Tree
- Recursive sorting algorithms
- Efficient searching algorithms
- Interactive menu system with ENUM
- File I/O for data persistence

## Technologies
- Java
- NetBeans IDE

## Project Structure
src/CA_2/
├── Manager.java (abstract parent class)
├── Department.java (abstract parent class)
├── Employee.java (abstract parent class)
└── ... (subclasses and supporting files)


## How to Run
1. Open project in NetBeans
2. Run BankSystem.java
3. Follow the interactive menu
